## Учебный проект по созданию микросервисной архитектуры.Тема проекта: магазин спорттоваров.
### Используемые технологии: Java Spring, Postgresql, Docker, Apache Kafka, Swagger UI, Keycloak
### Архитектура состоит из:
#### Api Gateway, созданный с помощью Keycloak
#### OrderService - сервис обработки заказов
#### PaymentService - сервис оплаты заказов
#### DeliveryService - сервис доставки заказов
#### Основная база данных - postgresql, поднятая в Docker
